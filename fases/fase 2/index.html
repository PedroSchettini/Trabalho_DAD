O que é uma branch?

Uma branch é uma linha de desenvolvimento separada da principal (geralmente chamada de "main" ou "master"), onde você pode trabalhar funcionalidades ou correções sem afetar diretamente o código principal. É como uma cópia do projeto onde você pode experimentar novas ideias.


Depois de fazer as alterações em uma branch e testá-las, você pode mesclá-las de volta na branch principal.

Próximo

Para criar uma branch digite o seguinte comando:

"git branch <nomedaBranch>"


(Coloque o nome da branch como "gitinho")

aluno-legal@kksjsskjs MINGW64 ~/iniciando_repositorio_gitinho

$

aluno-legal@kksjsskjs MINGW64 ~/iniciando_repositorio_gitinho(gitinho)

$
Próximo

Agora que aprendemos como criar uma branch, vamos voltar para a main para continuarmos a nossa aula. Digite o seguinte comando

"git checkout <nomedaBranch>"


Lembrando que vamos para a gitinho, ent coloque o nome da branch como "gitinho"

aluno-legal@kksjsskjs MINGW64 ~/iniciando_repositorio_gitinho(main)

$

aluno-legal@kksjsskjs MINGW64 ~/iniciando_repositorio_gitinho(gitinho)

$
Próximo

Agora para podermos pegar as alterações feitas dentro da branch nós iremos usar o seguinte comando:

"git pull origin main"

Comandos gerais

Aqui estão alguns exemplos de comandos gerais importantes do git:

"git init" -> inicializa um repositório local

“git status” -> exibe o estado atual do repositório, incluindo arquivos modificados e não rastreados.

“git add .” -> prepara todos os arquivos para salvar

“git commit -m “<mensagem>” ” -> salvar alterações

“git push origin <nomeDaBranch>” -> envia para o repositório remoto

“git clone <linkRepositorio>” -> clona um repositório existente do GitHub para um novo diretório local.
